# Getting started with the DeepSeek-optimized codex-ds fork

This repository is a fork of OpenAI Codex CLI that is specifically tuned for
DeepSeek-compatible providers.

If you are starting from this fork, read these pages first:

- [Installing & building](./install.md)
- [Configuration](./config.md)
- [Sample configuration](./example-config.md)

For a broader overview of upstream Codex CLI concepts, you can still reference
the official overview at
[developers.openai.com/codex/cli/features](https://developers.openai.com/codex/cli/features#running-in-interactive-mode),
but this fork's recommended setup path is the DeepSeek-oriented configuration
documented here in-repo.
